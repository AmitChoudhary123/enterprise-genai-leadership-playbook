# AI Solution Review Checklist

## Business fit

- Clear business owner
- Measurable outcome
- Workflow integration point identified

## Architecture

- Context sources approved
- Model/provider choice justified
- Integration and fallback path defined

## Evaluation

- Test set exists
- Quality metrics defined
- Hard gates identified

## Governance

- Risk tier assigned
- Human approval rules defined
- Audit and monitoring approach documented

## Adoption

- User workflow impact understood
- Training and change plan drafted
- Feedback loop defined
