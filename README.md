# Enterprise GenAI Leadership Playbook

A practitioner field guide for leaders turning GenAI, RAG, and agentic AI from scattered pilots into governed enterprise capability.

This repository is designed as a public leadership asset: part strategy playbook, part architecture review kit, part operating model toolkit. It is written for enterprise reality: business ownership, knowledge quality, evaluation discipline, governance, adoption, and repeatable delivery.

## Why this exists

Model access is no longer the differentiator. The differentiator is the enterprise system around the model: how use cases are selected, how context is engineered, how prompts and agents are controlled, how quality is evaluated, how risk is governed, and how teams learn after launch.

## Editorial stance

This is not a collection of definitions. Each asset should help answer one of five enterprise questions:

1. What AI work is worth funding?
2. What context and controls are required?
3. How do we know the system is good enough?
4. Who owns the workflow after launch?
5. What must be reusable for the next team?
