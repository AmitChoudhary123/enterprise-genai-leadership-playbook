# Publication Note

This repository was published as a living field guide rather than a finished book.

The intent is to make enterprise AI delivery thinking visible in public: frameworks, templates, operating model choices, governance patterns, and lessons learned that are usually trapped in internal decks or consulting artifacts.

## Why the repository is new

The repo is intentionally new and public. I am not trying to imply a long commit history. The value should come from the usefulness, clarity, and maturity of the material, not from artificial age.

## How it will evolve

- Version 0.1: Core executive playbook and reusable templates
- Version 0.2: Industry-specific examples and review packs
- Version 0.3: Workshop canvases and maturity assessment tools
- Version 0.4: Companion implementation examples connected to agent and evaluation repositories
- Version 1.0: Cohesive enterprise GenAI operating model toolkit

## Editorial standard

Every addition should pass three tests:

1. Would this help a senior leader make a better AI decision?
2. Would this help an architect or delivery lead avoid a real failure pattern?
3. Would this still be useful after the current model hype cycle changes?
